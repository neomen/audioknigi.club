# Audioknigi.club
Google Chrome extension for controlling Audioknigi.club player

## Installation

```bash
# clone it
$ git clone https://github.com/neomen/audioknigi.club.git

# Install dependencies
$ npm install
```

## Development

* Run script
```bash
# build files to './build'
# start gulp watch
$ npm start
```

## Build

```bash
# build files to './build'
$ npm run build
```
